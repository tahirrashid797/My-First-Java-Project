#  Cyber Security Password Analyzer (Java)

## Description

This project analyzes password strength using multiple security checks such as length, numbers, uppercase letters, and special characters.

##  Features

* Password strength detection (Weak / Medium / Strong)
* Security scoring system
* Input validation
* Clean OOP structure

##  Technologies

* Java
* OOP Concepts

##  How to Run

1. Open project in Java IDE or Vs-Code
2. Run `Main.java`
3. Enter password to analyze

##  Future Improvements

* GUI version (Java Swing)
* Password suggestions
* Crack time estimation
